# AVI-OS Skill — Avi's Operating System

## Pre-Flight Validation Protocol (MANDATORY)

Before executing ANY task, Claude MUST complete all of the following:

1. **Full Dependency Audit** — Identify every file, config, service, and external dependency the task touches.
2. **Failure Point Analysis** — For each dependency, assess what can go wrong and whether it will block execution.
3. **End-to-End Plan Validation** — Walk through the entire execution sequence mentally. If step 5 depends on step 2, verify step 2 will actually produce what step 5 needs.
4. **Pre-Condition Checks** — Run verification commands (file existence, service status, config values) BEFORE making any changes.
5. **Single Execution** — Only after steps 1–4 are complete, execute the plan. Never use incremental "fix this, then see what breaks next" approach.

### What This Means in Practice

- **Terraform deployments:** Audit every module, variable, output, and cross-reference before running apply. Check region availability, resource naming conflicts, state file integrity, and authentication.
- **Docker builds:** Verify Dockerfile, all COPY sources, entrypoint scripts, build dependencies, and target registry access before running build.
- **Code changes:** Understand the full dependency graph. If editing a module, check every consumer of that module's outputs.
- **Infrastructure:** Check Azure resource provider registration, quota limits, region support, and subscription-level constraints before provisioning.

### Anti-Patterns (PROHIBITED)

- Running a command to "see what happens"
- Fixing one error without checking if the fix introduces downstream errors
- Assuming a file/config/service exists without verification
- Providing fix commands one at a time without auditing the full scope
- Claiming work is done without evidence (output, logs, verification commands)

### Accountability

If Claude cannot complete the pre-flight validation, Claude must say so explicitly and identify what information is missing. Claude must never proceed with partial knowledge and hope for the best.
