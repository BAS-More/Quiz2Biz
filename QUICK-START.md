# Quiz-to-build (Quiz2Biz) - Quick Start Guide

## 📖 What is This Software?

**Quiz-to-build** is a smart questionnaire system that helps businesses:
1. 📝 Answer questions about their technology and processes
2. 📊 Get scored on 7 technical dimensions
3. 📄 Automatically generate professional documentation packages
4. 🎯 Identify gaps and improvement areas

Think of it as a **"tech readiness assessment + automatic documentation generator"** in one platform.

## 🚀 Getting Started (5 Steps)

1. **Sign Up** (2 minutes)
   - Go to `/auth/register`
   - Enter name, email, password
   - Verify email

2. **Start Questionnaire** (30-50 minutes)
   - Click "New Questionnaire"
   - Answer questions in 5 sections
   - Save progress anytime

3. **Review Scores** (5 minutes)
   - View dashboard heatmap
   - Identify strong/weak areas

   - Go to Documents page
   - Click "Generate"
   - Download DOCX/PDF files

   - Export documents
   - Share with team/investors
   - Track improvements

## 🔑 Core Concepts Explained

### 1. Adaptive Questions
Questions change based on your previous answers. For example:
- If you say "Yes, we use CI/CD" → Get detailed CI/CD questions
- If you say "No CI/CD" → Get questions about deployment process

### 2. Scoring Dimensions
Your answers are scored across 7 categories:
- **Modern Architecture**: Cloud, microservices, APIs
- **AI-Assisted Dev**: AI tools, automation, code generation
- **Coding Standards**: Code quality, reviews, guidelines
- **Testing & QA**: Test coverage, automation, types
- **Security**: DevSecOps, vulnerability management
- **Workflow**: CI/CD, deployment, monitoring
- **Documentation**: Technical docs, knowledge sharing

### 3. Evidence Registry
Link proof/evidence to your answers:
- Upload files, screenshots, logs
- Connect GitHub/GitLab repos
- Link Jira tickets, Confluence docs
- Track compliance requirements

### 4. Document Generation
Auto-creates professional docs from your answers:
- Uses templates with your data
- Includes scores and recommendations
- Export as DOCX or PDF
- Customize before download
